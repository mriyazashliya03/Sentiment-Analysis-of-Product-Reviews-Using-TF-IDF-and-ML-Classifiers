# Sentiment-Analysis-of-Product-Reviews-Using-TF-IDF-and-ML-Classifiers
 This repository implements a sentiment classifier for Amazon/Yelp reviews using TF-IDF (unigrams/bigrams) and engineered features like negation presence and punctuation density. We compare Logistic Regression, Naive Bayes, SVM, and Gradient Boosting, utilizing Chi-Square selection and reporting Macro F1 and PR metrics.
